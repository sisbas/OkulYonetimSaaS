# Demo Route Registry

| Route | Amaç |
| --- | --- |
| `/demo/today` | Günlük operasyon sunum akışı |
| `/demo/schedule` | Taslak/yayınlanmış program, conflict ve event modalı |
| `/demo/leave/:id` | İzin etki analizi ve yedek öğretmen seçimi |
| `/demo/attendance/session/:id` | Sentetik öğrenci yoklama etkileşimi |
| `/demo/notifications` | İnsan onaylı bildirim simülasyonu |

Tüm route'lar aynı deterministik statik fixture setini kullanır.
