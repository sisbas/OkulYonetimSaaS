# Demo Security Boundary

Bu uygulama yalnız sunum amaçlıdır.

- Gerçek kullanıcı kimliği yoktur.
- Auth, token, cookie ve permission catalog bağlantısı yoktur.
- API isteği veya harici ağ çağrısı yoktur.
- Gerçek öğrenci, veli veya guardian kişisel verisi yoktur.
- Telefon hedefleri maskelenmiş sentetik gösterimlerdir.
- Etkileşim state'i kalıcı değildir ve sayfa yenilenince sıfırlanır.
