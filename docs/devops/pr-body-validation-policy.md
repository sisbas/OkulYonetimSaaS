# PR Body Validation Policy

PR body makine tarafından doğrulanmalıdır.

## Zorunlu alanlar

- Amaç
- Kapsam
- Kapsam dışı
- Acceptance criteria
- Test çıktısı
- KVKK/audit etkisi
- Rollback
- CI run referansı

## Kural

Bu alanlardan biri boş, placeholder veya belirsiz ise PR Governance workflow'u fail olmalıdır.

## Kabul kriteri

PR açıklaması sadece niyet metni değil, merge kararında kullanılabilecek kanıt içermelidir.
