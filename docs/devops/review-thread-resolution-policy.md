# Review Thread Resolution Policy

Review thread resolution required olmalıdır.

## Kural

Açık inline review thread veya unresolved conversation varken PR merge edilememelidir.

## Gerekçe

Çözülmemiş review thread'leri açık teknik veya güvenlik itirazı anlamına gelebilir. Bunlar kapatılmadan merge yapılması governance açığıdır.

## Kabul kriteri

Tüm review thread'leri resolved olmadan merge butonu kullanılabilir kalmamalıdır.
